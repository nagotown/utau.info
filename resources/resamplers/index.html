    Resampler Directory

    The audio examples provided are using the voicebank Hitsuboku Kumi Act 4. UST by 番煎P

    Official Resamplers

    Resamplers created by Ameya.

    resampler

    Comes with each UTAU version.

    doppeltler

    Remake of resampler.

    • 009 / Tweet / DL
    • 008 / DL
    • 007 / DL
    • 006 / DL
    • 005 / DL
    • 004 / DL

    fresamp

    Stands for frameresampler.

    • 011 / Direct DL

    • 012 / Private

    • 013 / Private

    • 014 / Direct DL

    f2resamp

    Remake of fresamp.

    • 005 / Tweet / Direct DL
    • 004 / Tweet / Direct DL
    • 003 / Tweet / Direct DL
    • 002 / Tweet / Direct DL
    • 001 / Tweet / Direct DL

    phavoco

    Stands for phasevocoder.

    • 010 / DL

    phaavoco

    Remake of phavoco. Originally was going to be named phävoco/phaävoco, but Ameya opted for phaavoco to prevent filename issues.

    • 001 / Tweet / Direct DL

    model4

    A beta model that was only distributed to certain people. Makes random notes significantly quieter. Creates .frc files.

    Independent Resamplers

    Resamplers created by the community.

    moresampler

    By Kanru Hua. Can also be used as a wavtool.

    • 0.8.4 / BowlRoll
    • Website (Archived)

    macres

    By titinko. Crossplatform resampler.

    • GitHub

    TIPS

    By ScientistB. Good with soft voicebanks.

    • Homepage

    bkh01

    By Zteer.

    • 0.55

    VS4U (VocalShifter4UTAU)

    By Ackie Sound.

    • Homepage Scroll down to UTAU向けエンジン VS4U and click ダウンロード.

    presamp

    By Delta. Lets CV USTs process as CV, VCV, or CVVC when resampling. Can be used with whatever resampler or wavtool you want. Sometimes, config files for presamp will be included with non-Japanese banks to improve ease of use.

    • Homepage
    • English setup guide by KLAD
